# 📝 To-Do List Application

## 📌 Overview

This is a simple **To-Do List application** that helps users manage their daily tasks.
The application allows users to add, view, and delete tasks, making it easier to organize and track activities.

This project was created using **Visual Studio Code** and basic web technologies.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete completed tasks
* 💾 Simple and clean interface
* ⚡ Lightweight and easy to use

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Functionality and task management

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Rajat289/TO-DO-List.git
```

2. Open the project folder

```
cd todo-list
```

3. Open **index.html** in your browser.

---

## 📖 Usage

1. Enter a task in the input field.
2. Click the **Add** button to add it to the list.
3. Click the **Delete** button to remove a task from the list.
